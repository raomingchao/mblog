Sublime Text 使用技巧

发表于 2015-11-11   |   分类于 开发工具   |  

定制你的 Sublime 主题

  • 安装Package Control

安装 Material Theme

  • 打开控制台：shift+command+p
  • 选择Package Ctonroller: package install（等待loading完成)
  • 再次打开控制台->输入：material theme
  • 复制粘贴以下 到 settings-user -> command+s

安装 Colorsublime

  • shift+command+p -> install -> Enter
  • colorsublime -> Enter -> shift+command+p -> install Theme -> 选择一个color(ex. facebook)

修改项目控制板颜色

  • shift+command+p -> install -> Enter
  • shift+command+p -> package resourceViewer -> Enter
  • shift+command+p -> packageResourceViewer: extract package -> material Theme
  • 然后编辑 Preferences -> Browse Packages -> Material Theme -> Material-Theme.sublime-theme 文件中

选项设定

可以把default的选项覆盖到user上面去

  • 不显示行数字：”line_numbers”: false
  • 文本左边边距：”margin”: 0,
  • 安装专用代码字体：FiraCode -> “font_face”: “Fira Code”
